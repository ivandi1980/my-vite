# My Vite

## Description
`My Vite` is a simple Vite project template, which is based on `ReactJS`

## Installation
```bash
git clone https://github.com/ivandi1980/my-vite.git
```

## Usage
```bash
cd my-vite
npm install
npm run dev
```
